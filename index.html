🚀 如何拥有你的主页？

✨ 模式 A：统一模板（推荐，99% 主播）

只需配置数据文件，自动套用清新模板，支持歌单/视图切换/随机选歌等功能。

无需复制 index.html，系统自动加载统一模板

🎨 模式 B：完全自定义（高级，1% 主播）

启用开关 + 编写 custom.html，完全自由设计页面，零模板限制。

info.json 中添加 "use_custom_page": true

📋 模式 A：统一模板注册步骤

  1. Fork 本仓库 → suki-live/suki-live
  2. 在 /u/ 下新建你的目录，如 /u/miku/
  3. 创建 info.json + playlist.json（参考下方模板）
  4. （可选）添加 avatar.png / bio.md / favicon.ico
  5. 在 /u/registry.json 中添加你的信息
  6. 提交 Pull Request，标题格式：[Register] your-id
  7. 等待管理员审核（通常 24-48 小时）
  8. 合并后访问 https://miku.suki.live ✨
🍴 Fork 仓库 📋 查看主播名录
👉 先看看示例：example.suki.live

🎨 模式 B：完全自定义（简要）

  1. 同上创建 /u/your-id/ 目录
  2. 创建 info.json，添加 "use_custom_page": true
  3. 编写 custom.html（完全自定义页面）
  4. （可选）添加 custom.js 实现交互
  5. 提交 PR，合并后访问 https://your-id.suki.live

💡 详细文档见 README.md

✨ 页面支持

  • 🎵 歌单展示（支持网易云 / B 站 / YouTube 链接）
  • 🔄 视图切换（列表/卡片模式 + 偏好记忆）
  • 🎲 随机选歌（弹窗确认 + 无链接提示）
  • 📝 Markdown 自我介绍（支持标题/加粗/链接/分割线）
  • 🖼️ 头像自动加载（放置 avatar.png 即可）
  • ✉️ 匿名信集成（dreamail.cn，保护隐私）
  • 📱 响应式设计（移动端自动适配）
  • 🎨 清新淡色风格（CSS 变量统一管理）
